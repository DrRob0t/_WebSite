# Website Redesign TODO List

**Based on:** External feedback from SWOT Analysis and New Design Analysis  
**Created:** February 3, 2026  
**Status:** Planning Phase

## Overview

This document tracks all recommended changes based on external feedback to transform the Hyve Dynamics website from a technically credible but horizontally-positioned site to one that clearly communicates economic value and deployment-first thinking.

**Core Design Principle:**  
> One sensing platform. Multiple deployment states. Increasing operational permanence.

- **P1** - High (Structural Changes) - Important for messaging clarity

- **P3** - Low (Polish) - Nice to have refinements
